# Route_Builder

Маршрутизатор для поиска рейсов между городами по API Яндекс.Расписаний с кэшированием результатов и графическим интерфейсом на Qt.

Проект состоит из двух частей - двух возможных вариантов получения результата:

- **Консольное приложение** — спрашивает у пользователя API‑ключ, города отправления и прибытия, дату и тип маршрута (только прямые / прямые + с пересадками), затем выводит найденные рейсы в консоль.
- **GUI‑клиент** — Qt-приложение с окном MainWindow, использующее общий бэкенд для работы с API и кэшем.
